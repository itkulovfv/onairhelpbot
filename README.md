# OnAirHelpBot 📸

Telegram Mini App для управления фотографиями в разделах «Инвестор» и «Трейдер» на сайте Tilda.

## Особенности
- **Безопасность**: Все ключи хранятся на бэкенде. Проверка `initData` через HMAC-SHA256.
- **Белый список**: Доступ только для сотрудников с указанными Telegram ID.
- **Разделы**: Отдельное управление для «Инвестор» и «Трейдер».
- **Массовая загрузка**: Загрузка ровно 3 фото за раз.
- **Выборочное удаление**: Возможность выбрать конкретные фото для удаления.

## Технологии
- **Backend**: Python (aiohttp, python-telegram-bot)
- **Frontend**: HTML/JS (Telegram Web App)
- **Хранение данных**: Google Sheets (через Apps Script)
- **Хостинг фото**: ImgBB API

## Настройка
1. Скопируйте `.env.example` в `.env` и заполните данные.
2. Деплой бэкенда (`bot.py`) на **Railway**.
3. Деплой фронтенда (`index.html`) на **GitHub Pages**.
4. Настройка Google Sheets через `code.gs`.

Подробная инструкция по шагам находится в коде и документации.
